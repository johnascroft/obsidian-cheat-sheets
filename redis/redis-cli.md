# Redis CLI

Full list of commands can be found in the [Redis Commands](https://redis.io/commands)

To start up the Redis CLI in [Linux](linux):

```bash
redis-cli
```

To monitor what is going on in Redis, you can use:

```bash
redis-cli monitor
```
